Wirtualny Świat (C++)
Projekt zrealizowany w ramach przedmiotu Programowanie Obiektowe na Politechnice Gdańskiej.

Główne Funkcjonalności

System Turowy: W każdej turze wszystkie organizmy wykonują akcje zgodne ze swoim rodzajem.

Mechanika Walki i Kolizji: Organizmy mogą walczyć o pole, bazując na swojej sile i inicjatywie.

Człowiek: Sterowany przez gracza za pomocą strzałek na klawiaturze, posiadający umiejętność specjalną.

System Rozmnażania i Rozprzestrzeniania: Zwierzęta mogą się rozmnażać, a rośliny zasiewać nowe pola.

Zapis i Odczyt: Możliwość utrwalenia stanu świata w pliku.
